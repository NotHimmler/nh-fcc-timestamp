nh-fcc-timestamp

This is a simple web API for converting date types.

Pass in either a unix timestamp or a natural date string in the format "MONTH DAY_NUM, YEAR" and get back a JSON object with parameters representing both.

If it contains neither, it will return a JSON object with null for both paramaters.

Examples

URL

http://nh-fcc-timestamp.herokuapp.com/February 29, 2016
http://nh-fcc-timestamp.herokuapp.com/1456704000

Result

{ "natural": "February 29, 2016", "unix": 1456704000 }
